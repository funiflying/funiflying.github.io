markdown 基本語法

概述

markdown 的目標是易讀易寫
，markdown 撰寫的文件可以以純文本發佈，並且看起來不會由許多標籤和格式指令構成

markdown的目標是成為一種適用網絡的書寫語言，markdown並不是要取代html，甚至沒有什麼相近之處。markdown 是一種書寫格式，而html是一種發佈格式。
當然，在markdown 直接使用html標籤也是允許的。需要注意的是，在html標籤內使用markdown語法是無效的

列表

markdown支持無序列表和有序列表

無序列表使用星號、加號、減號作為列表標記

  • html
  • css
  • javascript
    有序列表使用數字加英文點號作為列表標記
  1. html
  2. css
  3. javascript

  4. This is a list item with two paragraphs. Lorem ipsum dolor
    sit amet, consectetuer adipiscing elit. Aliquam hendrerit
    mi posuere lectus.

    Vestibulum enim wisi, viverra nec, fringilla in, laoreet
    vitae, risus. Donec sit amet nisl. Aliquam semper ipsum
    sit amet velit.

  5. Suspendisse id sem consectetuer libero luctus adipiscing.

如果行首恰好出現類似列表項的格式，為了避免被識別為列表項，可以使用句点前面加上反斜杠
